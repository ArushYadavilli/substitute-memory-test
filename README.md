# ISEF-memory-test
This is a memory test for people who signed up to be a part of out ISEF study of how an offset of sleep schedule with an individuals chronotype, or natural sleep schedule. This is intended to be taken weekly as a part of a weekly cognitive assessment as part of our study. Feel free to fork the repository to personalize it, but no pull requests will be accepted. 
Webpage link: [https://arushyadavilli.github.io/substitute-memory-test/]((https://arushyadavilli.github.io/substitute-memory-test/))
